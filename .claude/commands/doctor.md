Run the asimov doctor command to diagnose project health.

```bash
asimov doctor
```

This performs comprehensive health checks:
- Protocol files validation
- Roadmap structure verification
- Project configuration checks
- Git hooks status
- License detection
- Claude Code settings verification

Review the output and address any issues flagged. Green checkmarks indicate healthy components, warnings indicate areas needing attention.
